Multiple Marks
==============
This example demonstrates creating a single chart with multiple markers
representing the same data.


.. altair-plot::
    :code-below:

    import altair as alt
    from vega_datasets import data

    stocks = data.stocks()

    alt.LayerChart(stocks).encode(
        x='date:T',
        y='price:Q',
        color='symbol:N'
    ).add_layers(
        alt.Chart().mark_point(),
        alt.Chart().mark_line()
    )
